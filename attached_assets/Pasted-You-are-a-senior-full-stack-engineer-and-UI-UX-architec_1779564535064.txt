─────────────────────────────────────────────
PAGES & FEATURES
─────────────────────────────────────────────
1. Home Page
   - Full-screen hero with CTA ("Get a Free Quote")
   - Stats bar: years of experience, projects completed, team size, countries
   - Featured services (3–4 cards)
   - Featured projects (3 cards with overlay)
   - Testimonials carousel
   - Call-to-action banner

2. About Page
   - Company story + mission statement
   - Core values section (grid icons)
   - Team members section
   - Certifications / Awards

3. Services Page
   - Service cards grid:
     · Residential Construction
     · Commercial Construction
     · Renovation & Remodeling
     · Interior Design
     · Project Management
     · Structural Engineering
   - Each service: icon, title, short description, "Learn More" button

4. Projects Page
   - Filterable portfolio grid (by category: Residential / Commercial / Industrial)
   - Each project card: image, title, category badge, location, year
   - Project detail modal or page

5. Contact Page
   - Contact form (name, email, phone, project type, message)
   - Form handled by Redux + Axios → Express API → Nodemailer
   - Embedded Google Maps placeholder
   - Contact info cards (address, phone, email)

6. Global Components
   - Sticky responsive Navbar with mobile hamburger menu
   - Footer with links, social icons, newsletter signup
   - Scroll-to-top button
   - Loading spinner
   - 404 Not Found page

DESIGN SYSTEM
─────────────────────────────────────────────
Aesthetic: Industrial & Authoritative
  - Primary: #F5A623 (construction amber/orange)
  - Dark: #1A1A1A (near-black)
  - Neutral: #F5F5F0 (off-white concrete tone)

  - Heavy use of bold typography, strong grid lines, sharp geometry
  - Subtle concrete texture on hero and section backgrounds
  - Smooth scroll animations using Framer Motion (stagger reveals)

─────────────────────────────────────────────
BACKEND API ENDPOINTS
─────────────────────────────────────────────
GET    /api/projects          → list all projects
GET    /api/projects/:id      → single project
GET    /api/services          → list all services
POST   /api/contact           → submit contact form (sends email)
GET    /api/stats             → company stats (years, projects, etc.)

─────────────────────────────────────────────
STATE MANAGEMENT (Redux Toolkit)
─────────────────────────────────────────────
Slices:
  - projectsSlice   → fetchProjects, filter by category, selectedProject
  - servicesSlice   → fetchServices
  - contactSlice    → form state, loading, success, error
  - uiSlice         → mobileMenu open/close, scrolled state
